# EduVolveBD

EduVolveBD is a smart and interactive e-learning platform designed to improve the online learning experience for students, instructors, and administrators. The platform provides a structured environment for course management, quizzes, assignments, progress tracking, and student engagement through gamification features.

This project was developed as a Final Year Design Project (FYDP) for the Bachelor of Science in Computer Science and Engineering (CSE).

---

## Features

### Admin Features
- Manage students and instructors
- Manage courses and categories
- Monitor platform activities
- Control user access and permissions
- Manage system settings

### Instructor Features
- Create and manage courses
- Upload learning materials
- Create quizzes and assignments
- Evaluate student performance
- Track student progress

### Student Features
- Enroll in courses
- Access learning materials
- Submit assignments
- Participate in quizzes
- Track learning progress
- Earn badges and achievements

## Installation

### Clone the Repository

```bash
git clone https://github.com/your-username/EduVolveBD.git
```

### Move to Project Directory

```bash
cd EduVolveBD
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Virtual Environment

#### Windows

```bash
venv\Scripts\activate
```

#### Linux / MacOS

```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Database Migrations

```bash
python manage.py migrate
```

### Start Development Server

```bash
python manage.py runserver
```
